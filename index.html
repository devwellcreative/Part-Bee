PartBee Components

PartBee is composed by 3 groups of components: Layout Components, Responsive Structures and Plugins.

Layout Components

Those that help a build base components of layout like header, navigation menu, sections, featured banner and footer

Responsive Structures

These are useful components. They are more a big structure to make your job easier.

Plugins

These are plugins that I created to attend something.

Using it

Partbee is totally fractionated. Every component has your own CSS file and dependencies. The common frameworks (Bootstrap, jQuery, font Awesome and others) are the same references in vendor folder. In the default-structure.css file you can customize base components following your layout.

To use a component add css file for that component on html document.

<link rel="stylesheet" href="content/styles/default-component-name.css" />

Use the template named by the name of component in the part folder as sample. It’s importante keep the mainly structure for the component, your style is totaly dependent of this.

<!-- PARTBEE -->
<center>
<i>Part Bee</i>
</center>
